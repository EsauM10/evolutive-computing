# Evolutive Computing
## Image Evolution
Algoritmo genético aplicado ao processo de evolução de uma imagem
- As imagens são representadas por matrizes binárias 13x13
- Cada bit da matriz é mapeado para um emoji (0-⬛️, 1-⬜️)
